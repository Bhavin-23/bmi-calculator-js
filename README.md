# BMI Calculator

This is a responsive BMI (Body Mass Index) Calculator built using **HTML**, **CSS**, **JavaScript**, and **Bootstrap**.

## 📝 Description

The app allows users to input their **weight (in kg)** and **height (in meters)**, then calculates their BMI with a result indicating whether they are:
- Underweight ❗
- Normal ✅
- Overweight ⚠️
- Obese 🚨

It provides instant feedback using color-coded messages.

---

## 🚀 Features

- User-friendly interface
- Validates user input
- Calculates BMI using formula:  
  \[
  \text{BMI} = \frac{\text{weight (kg)}}{\text{height (m)}^2}
  \]
- Displays result message with category
- Responsive layout using Bootstrap
